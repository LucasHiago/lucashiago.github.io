## Quick start

#To install

```sh
npm install
```
#To start
```sh
npm start
```
#To build
```sh
npm run build
```

#To delploy gh-pages
```sh
npm run deploy
```

#Technologys

- [x] Node.js
- [x] React.js
- [x] Chart.js
- [x] Bootstrap.css
- [x] API GOOGLE MAPS
- [x] Class Names
- [x] HashRoute


Quick start options:

- Clone the repo: `git clone https://github.com/LucasHiago/lucashiago.github.io.git`.
